## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

    NodeJS

### Setup

    1. git@github.com:jevazquezb/Todos_app_react.git
    2. cd Todos_app_react

### Install

    npm install

### Usage

    npm start
